# Caso de datos bencinera 2022
Integrantes:
Jean Pierre Urquiaga Vidal

Vicente Ignacio Yunusic Salinas

El dataset representa el registro telemétrico e histórico de transacciones de combustible de una flota industrial mixta durante el año de 2022. Dicha data nos muestra el consumo interno de vehículos y maquinarias pesadas que operan en 10 sectores productivos distintos, siendo los principales: Minería, Operaciones Petroleras, Transporte y Construcción.

En el presente repositorio se adjuntan los archivos para segundo encargo de PROGRAMACION PARA LA CIENCIA DE DATOS 001D
